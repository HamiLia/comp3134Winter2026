SCENARIO 3: Navigate to the FORM and type: <script>alert(%27hello%27)</script>
==============================================================
Text entered in form: <script>alert(%27hello%27)</script>

Observation:
No JavaScript alert popup appears. Dictionary.com sanitizes the input before
processing it. The site either strips the script tags entirely or encodes them
as HTML entities so they are treated as plain text, not as executable code.
This is an example of proper XSS (Cross-Site Scripting) mitigation on a
well-implemented production website.
